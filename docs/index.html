After sequencing, expression was quantified by counting the number of UMIs mapped to each gene using scPipe(Tian et al. 2018). Count data for all endogeneous genes (GENCODE Release 28 (GRCh38.p12)) and spike-in transcripts (ERCC) are available in this repository.

This analysis is adapted from the Orchestrating Single-Cell Analysis with Bioconductor book and the RNA-seq analysis is easy as 1-2-3 with limma, Glimma and edgeR workflow (Law et al. 2016).
